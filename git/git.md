# What is Git?
Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency. Our team uses it to manage and interact with our many projects on Github.
# How Can I Learn It?
Git is not intuitive to learn as Java. You really just have to learn it as you go along. As you need it, either ask another subteam member, or look it up on a website like [StackOverflow](https://stackoverflow.com/).
